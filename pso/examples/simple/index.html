Instructions

Carry out the following experiments and answer the questions on the Google form. Be sure to refresh the page before carrying out each of these experiments to reset the results from the previous experiment.

  1. Set function to optimze to be Rastrigin, and run the simulation 5 times.
  2. Set function to optimze to be Rastrigin, social influence to 0 and run the simulation 5 times by clicking optimize to start each run. Wait for a value to be added to the list under best for each run before clicking optimize to start the next run.
  3. Set function to optimze to be Rastrigin, personal influence to 0 and run the simulation 5 times by clicking optimize to start each run. Wait for a value to be added to the list under best for each run before clicking optimize to start the next run.
  • The blue line indicates the global best solution.
  • The red lines indicate positions of particles.
